# Security Policy

Thank you for helping keep this repository secure.

## Supported Version

The `main` branch is the supported version of this project.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Send reports to:

```text
agarwalujala3@gmail.com
```

Include:

- A short description of the issue.
- Affected files, routes, components, or workflows.
- Clear reproduction steps.
- Expected impact.
- Any suggested mitigation, if known.

## Response Expectations

Security reports will be reviewed privately. Valid issues will be triaged and fixed as quickly as practical based on severity and project scope.

## Safe Testing Guidelines

Please avoid:

- Denial-of-service or stress testing against live deployments.
- Access attempts against accounts, credentials, or private infrastructure.
- Public disclosure before a fix is available.
